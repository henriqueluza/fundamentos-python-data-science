# Fundamentos de Python para Data Science

Este repositório contém material de estudo e prática sobre fundamentos de Python aplicados à ciência de dados.

## Sobre o Projeto

O conteúdo aqui apresentado tem como base os laboratórios e aulas do curso **IBM Data Science**, abrangendo conceitos essenciais e ferramentas práticas para análise de dados, visualização e machine learning.

## Conteúdo

O projeto está organizado em notebooks Jupyter que cobrem os seguintes tópicos:

- **Fundamentos-Python.ipynb**: Conceitos básicos da linguagem Python
- **Data-Analysis-Python.ipynb**: Técnicas de análise de dados com Python
- **Data-Visualization-Python.ipynb**: Visualização de dados
- **Pandas.ipynb**: Manipulação e análise de dados com Pandas
- **Plotly&dash-Python.ipynb**: Visualizações interativas com Plotly e Dash
- **Machine-Learning.ipynb**: Introdução ao Machine Learning com regressão linear
- **Python-Banco-de-Dados.ipynb**: Integração com bancos de dados

## Tecnologias Utilizadas

- Python 3
- NumPy
- Pandas
- Matplotlib
- Plotly
- Scikit-learn
- Jupyter Notebook

## Como Utilizar

1. Clone este repositório
2. Instale as dependências necessárias (NumPy, Pandas, Matplotlib, Scikit-learn, etc.)
3. Abra os notebooks Jupyter para explorar o conteúdo
